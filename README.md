# Başak Vardiya

Başak Kır Pidesi — 4 şube haftalık vardiya yönetimi. Tek kod tabanı; **web**,
**Windows (.exe)** ve **Android (APK)** olarak çalışır. React + Vite + TypeScript
+ Firebase (Firestore + Auth), Electron, Capacitor.

Kurulum, seed, build ve dağıtım adımları için **[KURULUM.md](./KURULUM.md)**.

```bash
npm install
cp .env.example .env   # Firebase değerlerini doldurun
npm run seed           # 4 şube ön ayarları + örnek hafta
npm run dev            # http://localhost:5173
```
